# PipingDesignKit

配管揚程圧損計算ツール — 「根拠が見える圧損計算」

## 概要

ブラウザ上で動作する配管圧損計算ツール。計算結果だけでなく、使用した摩擦係数・管粗度・局部損失係数・流体物性の出典と値をすべて追跡可能にする。

- **Excel職人芸からの脱却** — 計算ロジックとパラメータの根拠をリポジトリで管理
- **ルート変更時の即時再計算** — 3Dルート入力で配管ルートを引き直せば圧損を自動再計算
- **図面と計算書の一致** — アイソメ・立面・平面の3ビューで確認しながら入力
- **GitHub Pagesで共有** — インストール不要、URLを渡すだけ

## プロジェクト構成

```
src/
├── domain/          … 計算ロジック（純粋関数、外部依存なし）
├── application/     … ユースケース
├── infrastructure/  … データソース読み込み、エクスポート
└── ui/              … React コンポーネント

data/
├── fluid-properties/   … 流体物性データ
├── pipe-specs/         … 配管仕様データ
├── fittings-db/        … 継手損失係数データ
└── references/         … 出典情報
```
